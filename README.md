#Defence Procurement & Spend Analysis - Tableau Dashboard

##Overview
This project is a Tableau dashboard created to analyze defence procurement.
spend acrossservices, vendors, and equipment categories.

##Tools Used
-Tableau
-CSV Data

##Key Features
- KPI cars for total procurement spend
- Service-wise and vendor-wise analysis
- Top-N filtering
- Interactive dashboard
