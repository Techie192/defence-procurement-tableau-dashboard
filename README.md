# 🛡 Defence Procurement & Spend Analysis - Tableau Dashboard

## 📌 Project Overview
The **Defence Procurement & Spend Analysis Dashboard** is an interactive Tableau-based analytics project designed to analyze defence procurement expenditures across multiple services, vendors, and equipment categories.

This dashboard helps visualize procurement trends, identify high-spending sectors, analyze vendor contributions, and monitor expenditure distribution through interactive charts, KPI cards, and filters.

The project demonstrates data visualization, dashboard design, and analytical storytelling using Tableau.

---

# 🎯 Objectives
- Analyze defence procurement spending patterns
- Compare expenditures across defence services
- Identify top vendors and suppliers
- Understand category-wise procurement distribution
- Build an interactive and user-friendly dashboard
- Demonstrate business intelligence and data visualization skills
